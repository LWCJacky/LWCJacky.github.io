# 森林過度開發帶來的災害

能源與環境-USR 課程 SDGs項目之策略提案報告

# 目錄

  • Need(至少三張投影片)
  • SWOT (一張投影片)
  • Solution(至少三張投影片)
  • Differentiation(一張投影片)
  • Benefits(一張投影片)
  • 結論與討論(二張投影片)
  • 參考資料

# NEED 需求

!!

  • 齊柏林鏡頭下的台灣山林，水土保持失調導致土石流的發生已成為常態。

SDGs 目標15

  • 保育及永續利用陸域生態系，確保生物多樣性並防止土地劣化

# SWOT分析

  • S 全自動運作，減少人力。

# 解決方案

  • 利用低速定翼機進行巡航，構建地圖

    • SLAM：Simultaneous Localization and Mapping，即時定位與地圖構建技術
    • https://www.esentra.com.tw/2021/06/%E7%94%A8%E6%96%BC%E8%88%AA%E7%A9%BA%E6%B8%AC%E9%87%8F%E7%9A%84-lidar-%E8%88%87%E6%94%9D%E5%BD%B1%E6%B8%AC%E9%87%8F__%E5%B9%AB%E5%8A%A9%E6%82%A8%E9%81%B8%E6%93%87%E6%B8%AC%E9%87%8F%E5%82%B3%E6%84%9F/

  • 採用全自動化控制減少人力干涉

  • 利用影像分析估算出森林減少速度並劃出重點區域

  • 依照重點區域設立警戒區，採取應變措施

    • 惡意砍伐
    • 自然災害預防
    • 山老鼠

# 差異性比較

# 效益

  • 森林開發受到有效監控
  • 能立即掌握山林動態
  • 降低土石流發生機率
  • 偵測到異常情況，就能馬上得知並採取應變措施

# 結果與討論

# 結果

  • 利用系統，使山林破壞程度降低
  • 環境檢測系統有效偵測惡意砍伐，應證保育及永續利用陸域生態系

# 討論

  • 前期需大量人才資金，團隊是否能應付?
  • 旋翼機偵測範圍較小，需要以數量增加範圍
  • 旋翼機可定點盤旋，未來可與定翼機
